What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each element in the document (such as headings, paragraphs, or images) becomes a node within that tree.

How do you select an element from the DOM?

We can select an element in various way. Like, getElementById selects a single element by its unique ID and returns the first matching element or null if no match is found. getElementsByClassName selects all elements with the specified class name and returns a live HTMLCollection of matched elements. getElementsByTagName selects all elements with the given tag name and returns a live HTMLCollection. querySelector selects the first element that matches a CSS selector and returns the first matching element or null if no match is found. querySelectorAll selects all elements that match a given CSS selector and returns a static NodeList of matched elements. getElementsByName selects all elements with a specific name attribute and returns a live NodeList.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where instead of attaching event listeners to individual child elements, you attach a single event listener to a parent element. The parent listens for events triggered by its children and handles them using a process called event bubbling, where events triggered on a child element propagate (or "bubble up") to its ancestors.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using getAttribute() to retrieve, setAttribute() to update, and removeAttribute() to delete an attribute. You can also directly modify attributes like id, href, or src via dot notation (e.g., element.id = 'newId'). To modify styles, use element.style.property to change inline styles (e.g., element.style.backgroundColor = 'red') or classList.add(), remove(), or toggle() to modify classes. You can also use getComputedStyle() to access the final computed styles of an element.
